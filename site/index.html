Page 1 – Data Preprocessing and Preparation

Author: Insight Lab
Last updated: 2025-03-12

1.1 Data Acquisition and Format Conversion

Electrochemical Impedance Spectroscopy (EIS) measurements were obtained using Bio-Logic potentiostatic/galvanostatic instruments during lithium-ion cell characterization experiments. The raw impedance data, recorded in .mpr format, were converted to .xlsx format via EC-Lab’s export utility to enable Python-based analysis and model development.

Each dataset comprised five core impedance-domain features that comprehensively represent the electrochemical system’s frequency-domain response and served as the sole input space for all subsequent analysis stages:

  • Frequency (f, Hz)
  • Real Impedance (Z′, Ω)
  • Imaginary Impedance (Z″, Ω)
  • Impedance Magnitude (|Z|, Ω)
  • Phase Angle (θ, degrees)

1.2 Data Cleaning and Filtering

To ensure spectral integrity and eliminate measurement noise, the following preprocessing steps were applied to every exported spectrum:

  • Metadata removal: Instrument headers, operator comments, and non-numeric fields were excluded.
  • Noise filtering: Data points with negative impedance values or open-circuit anomalies were discarded.
  • Frequency validation: The operational frequency window was constrained to 10⁻² Hz – 10⁵ Hz for physical consistency.
  • Interpolation and smoothing: Minor gaps and irregularities were corrected to ensure monotonic frequency order and smooth Nyquist trajectories.

Each processed spectrum was visually validated via Nyquist and Bode plots to confirm continuity and physical plausibility before inclusion in downstream analysis.

1.3 Feature Normalization

All five features were normalized to a [0, 1] range using Min–Max scaling, preserving proportionality across frequencies while avoiding numerical dominance by large-magnitude impedance values. The resulting feature dictionary is summarised below.

Feature Symbol Unit Description
Frequency f Hz Excitation frequency for impedance measurement
Real Impedance Z′ Ω Resistive component of impedance
Imaginary Impedance Z″ Ω Reactive (capacitive/inductive) component
Impedance Magnitude |Z| Ω Total impedance magnitude, √(Z′² + Z″²)
Phase Angle θ ° Phase shift between voltage and current

This refined feature set provides a compact yet complete numerical description of each cell’s impedance response and underpins every subsequent modelling and evaluation phase.
