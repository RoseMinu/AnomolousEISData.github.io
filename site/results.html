Status: Ready for integration and publication

5.1 Summary

This work presents a comprehensive, data-driven EIS analysis framework that spans acquisition, processing, dimensionality reduction, and unsupervised evaluation of lithium-ion cell impedance spectra. The methodology demonstrates that a reduced feature set focusing on impedance domain variables yields physically interpretable and highly discriminative representations of cell behaviour.

  1. Standardised EIS data acquisition and preprocessing.
  2. Normalised feature extraction from (Z′, Z″, |Z|, θ, f).
  3. Dimensionality reduction using t-SNE, UMAP, and CNN Autoencoder embeddings.
  4. Unsupervised clustering and anomaly detection of electrochemical patterns.

5.2 Future Work

Next Step 1

Latent feature interpretation

Correlate learned representations with ageing indicators such as charge-transfer resistance and diffusion impedance.

Next Step 2

Temporal modelling

Extend the framework to time-dependent EIS sequences using time-gated LSTM or Transformer architectures.

Next Step 3

Operational integration

Fuse EIS embeddings with cycling performance data to enhance State-of-Health estimation and degradation prediction.
