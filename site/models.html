4.1 Dataset Splitting

The EIS dataset was partitioned into training, validation, and test cohorts to support reproducible experimentation and reliable generalisation assessment. Splits were stratified across frequency ranges, ensuring balanced representation from low-, mid-, and high-frequency domains.

70% Training set

Used for fitting dimensionality reduction, clustering, and anomaly models.

15% Validation set

Supports hyperparameter tuning and anomaly threshold calibration.

15% Test set

Reserved for final evaluation of clustering robustness and detection recall.

4.2 Evaluation Metrics

(a)

Clustering Metrics

  • Silhouette Score (S): Measures cluster cohesion and separation.
  • Davies–Bouldin Index (DBI): Lower values indicate better cluster distinctness.
  • Calinski–Harabasz Index (CHI): Evaluates the ratio of inter- to intra-cluster variance.
(b)

Autoencoder Performance

  • Mean Squared Reconstruction Error (MSE): Quantifies reconstruction fidelity.
  • Latent Space Variance: Measures information compactness and diversity.
(c)

Anomaly Detection

  • Precision–Recall (PR) & F1 Score: Assess accuracy in identifying anomalous spectra.
  • ROC–AUC: Quantifies separability between normal and abnormal impedance responses.
